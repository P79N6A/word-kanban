## Development

1. Clone & Init

	```
	$ git clone https://github.com/crispgm/word-kanban.git
	$ yarn install
	```

2. Migrate

	```
	$ node_modules/.bin/sequelize db:migrate
	```

3. Run

	```
	$ GOOGLE_TRANSLATE_API_KEY=your-api-key yarn start
	```
